iftop

mac安装iftop方式

  1. brew install iftop
  2. 在/usr/local/Cellar/iftop/下找到iftop的可执行文件，我的在/usr/local/Cellar/iftop/1.0pre4/sbin/iftop
  3. 将该文件软连接到/usr/local/bin目录下，ln -s /usr/local/Cellar/iftop/1.0pre4/sbin/iftop /usr/bin/local/iftop
  4. 执行时需要加上sudo
