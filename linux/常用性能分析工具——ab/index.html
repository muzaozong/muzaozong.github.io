阅读量:

简介

ab全称apache bench，是一款web压测工具

ab只能测试http请求，https的压测可以使用abs。(来自网络，我的机子可以正常测试https)

分析报告

以百度为例，命令如下

ab -t 10 -n 500 -c 10 https://www.baidu.com/

主要分为如下几个部分

请求进度

web服务器信息

请求的文档的相关信息

压测指标

  • Concurrency Level 请求并发数
  • Time taken for tests 测试持续时间
  • Complete requests 完成的请求数
  • Failed requests 失败的请求数
  • Total transferred 总的网络传输量
  • HTML transferred HTML内容的传输量
  • Requests per second 吞吐量
  • Time per request 用户平均请求等待时间
  • Time per request 服务器平均请求处理时间
  • Transfer rate 每秒网络上的平均流量
