top

top是linux中用于实时显示系统中各个进程资源占用状况的。

Linux的top命令使用出来是这样一个样子，

参数解释

第三行 %Cpu(s)

  • us 用户空间占比
  • sy 系统空间占比
  • ni 用户进程空间内改变过优先级的进程占用cpu百分比 (可以参考linux的nice命令)
  • id 空闲cpu占比
  • wa 等待IO的CPU占用
  • hi cpu处理硬件中断的时间
  • si cpu处理软件中断的时间
  • st 用于虚拟cpu的时间

第六行

  • PID 进程id
  • USER 运行用户
  • PR 优先级
  • NI 任务nice值
  • VIRT 进程使用的虚拟内存总量，单位kb。VIRT=SWAP+RES
  • RES 物理内存用量
  • SHR 共享内存用量
  • S 进程状态。S表示休眠，D表示不可中断的休眠，R表示运行，Z表示僵死，T表示停止或跟踪状态
  • %CPU 占用CPU的百分比
  • %MEM 占用的物理内存的百分比
  • TIME+ 累计CPU占用时间
  • COMMAND 该进程的命令名称。一行显示不下会进行截取
