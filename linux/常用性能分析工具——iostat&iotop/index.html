常用性能分析工具——iostat&iotop
发表于 2018-09-14| 更新于 2019-12-13|linux
字数总计: 406|阅读时长: 1 分钟| 阅读量:

iostat和iotop是用来监测磁盘IO的工具。

iostat

mac和linux的iostat略有不同

mac

不加参数

结果解释

  • cpu
    • us 用户占用百分比
    • sy 系统占用百分比
    • id 空闲百分比
  • load average
    • 1m 最近1分钟cpu的平均负载
    • 5m 最近5分钟cpu的平均负载
    • 15m 最近15分钟cpu的平均负载

有关load average可以参考 理解load average和cpu使用率

指定打印间隔时间

只显示设备数据

linux

不加参数

结果与mac端的排版差别挺大的

指定打印间隔时间

设置磁盘的读取单位

只打印磁盘

打印详细信息

指定磁盘

iotop

mac和linux的iotop也差别相当大

mac

无参数执行

不清除信息
