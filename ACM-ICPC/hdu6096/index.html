hdu 6096 String (AC自动机)
发表于 2017-08-12| 更新于 2019-12-02|ACM/ICPC
字数总计: 848|阅读时长: 4 分钟| 阅读量:

解法

解法见forever97的博客

主要就是将模式串以[后缀#前缀]的形式插入AC自动机中。然后匹配时将原串以[原串#原串]的形式去进行匹配。因为中间有字符’#’，所以不会出现多次同一模式串多次匹配的情况。最后通过fail链来累加计数就可以了。
