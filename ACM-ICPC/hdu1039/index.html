hdu 1039 Easier Done Than Said? (正则表达式)
发表于 2017-09-12| 更新于 2019-12-02|ACM/ICPC
阅读量:

解法

\\w*[aeiou]\\w*表示匹配元音
\\w*([aeiou]{3}|[^aeiou]{3})\\w*表示匹配连续3个元音或连续3个辅音
\\w*([^eo])\\1\\w*表示匹配相邻相同的字母(除o、e外)
