找到的规律如下：

通过观察，可以发现偶数行是有规律的
$f[2] = a[1] + a[2]$
$f[4] = a[1] - a[2] + a[3] - a[4]$
$f[6] = a[1] + a[2] + 2a[3] + 2a[4] + a[5] + a[6]$
$f[8] = a[1] - a[2] + 3a[3] - 3a[4] + 3a[5] - 3a[6] + a[7] - a[8]$
$f[10] = a[1] + a[2] + 4a[3] + 4a[4] + 6a[5] + 6a[6] + 4a[7] + 4a[8] + a[9] + a[10]$
如果不算上正负号，可以很容易发现前面的系数构成杨辉三角，也就是组合数，那这一题也就可以迎刃而解了。
对于奇数行只需要计算到下一行便可以变成偶数行来进行计算了。
