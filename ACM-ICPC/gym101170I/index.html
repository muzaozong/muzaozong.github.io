Gym 101170I Iron and Coal (搜索)
发表于 2017-07-08| 更新于 2019-12-02|ACM/ICPC
字数总计: 691|阅读时长: 3 分钟| 阅读量:

解法

其实仔细想想，这题的答案其实就是$min(dist_{t}+iron_{t}+coal_{t})$，其中$dist_{t}$表示$1$到$t$的最短距离，$iron_{t}$表示$t$到最近的铁矿的最短距离，$coal_{t}$表示$t$到最近的煤矿的最短距离。这个时候，你可能会想若是对于$t$而言，它到铁矿的最短路径和到煤矿的最短路径重合岂不是就会出现问题，但事实上并没有任何影响。举个简单的例子，假设煤矿在铁矿的路径上(反之同理)，设煤矿位置为$p$，那么不难想象$dist_{t}+iron_{t}+coal_{t} >= ans_{t} >= ans_{p} = dist_{p}+iron_{p}+coal_{p}$，而由于我们取的是最小值，所以我们最终的答案并不会受到影响。
